## 하락일 매수 규칙

매달 각 종목 배정액을 **기본풀 60% + 급락풀 40%**로 나눕니다.

- **기본풀**: 매 거래일 조금씩 꾸준히 매수 (평균 단가 만들기)
- **급락풀**: 많이 빠진 날 집중 투입

하락 기준:
- ETF: -1% / -2% / -3% 구간별로 보너스 매수 증가
- 개별주: -3% / -5% / -7% (변동성이 커서 기준을 넓게)

## 설치 — 30분, 순서대로만 따라오면 됩니다

### STEP 1. GitHub에 코드 올리기

1. https://github.com 접속 → 로그인
2. 우측 상단 `+` → `New repository`
3. 이름 입력 (예: `invest-bot`) → **Public** 선택 → `Create`
4. 생성된 페이지에서 `uploading an existing file` 클릭
5. 이 폴더의 파일 전부를 드래그해서 올리고 `Commit changes`

### STEP 2. 슬랙 준비 (알림 받을 곳)

1. 슬랙 앱 설치 후 워크스페이스 만들기 (예: "내 투자")
2. 채널 하나 생성 (예: `#매수알림`)
3. https://api.slack.com/apps → `Create New App` → `From scratch`
4. 앱 이름 짓고 워크스페이스 선택
5. 왼쪽 메뉴 `Incoming Webhooks` → 켜기(Activate)
6. `Add New Webhook to Workspace` → 채널 선택 → `Allow`
7. 생성된 **Webhook URL 복사** (https://hooks.slack.com/... 로 시작)

### STEP 3. GitHub에 슬랙 URL 등록

1. 아까 만든 GitHub 저장소 → `Settings` 탭
2. 왼쪽 `Secrets and variables` → `Actions`
3. `New repository secret` 클릭
4. Name: `SLACK_WEBHOOK_URL`
5. Secret: STEP 2에서 복사한 URL 붙여넣기 → `Add secret`

### STEP 4. 웹앱 켜기 (아이폰 확인용)

1. 저장소 → `Settings` → `Pages`
2. Source: `Deploy from a branch`
3. Branch: `main`, 폴더: `/ (root)` → `Save`
4. 몇 분 후 `https://[아이디].github.io/[저장소명]` 접속 가능
5. `index.html` 파일 열어서 상단 `GH_USER`, `GH_REPO`를 본인 것으로 수정

### STEP 5. 아이폰 홈 화면에 추가

1. Safari로 웹앱 주소 접속
2. 하단 공유 버튼 → `홈 화면에 추가`
3. 앱처럼 실행됨
